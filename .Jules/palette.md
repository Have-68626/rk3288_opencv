## 2024-06-17 - Keyboard Accessibility on Standalone Inputs
**Learning:** Adding `onPressEnter` to standalone text inputs improves keyboard accessibility for users. They expect to trigger associated actions using the Enter key, especially when dealing with single-input forms like person ID enrollment.
**Action:** When a `<Input>` is closely paired with a submit button, abstract the submit logic to a separate function and bind it to both the button's `onClick` and the input's `onPressEnter`.
