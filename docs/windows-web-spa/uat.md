# 用户验收测试（UAT）场景清单

## 场景 1：启动与访问
1) 双击运行 `win_local_service.exe`
2) 打开浏览器（Chrome/Edge/Firefox 任意一个）
3) 输入地址：`http://127.0.0.1:<port>/`
4) 预期：
   - 页面可打开
   - 顶部显示“后端已连接/不可用”的状态提示

## 场景 2：设置（本地偏好）
1) 进入“设置”页 → “应用（本地）”
2) 修改主题（亮色/暗色/系统）、语言（zh-CN/en-US）、启动页（概览/预览/设置）
3) 点击“保存本地偏好”
4) 预期：
   - 提示“保存成功”
   - 刷新页面后仍保持

## 场景 3：设置（后端配置写入 + 热生效）
1) 进入“设置”页 → “后端（API）”
2) 修改 `http.port`（或其它字段）
3) 点击“保存到后端”
4) 预期：
   - 返回成功提示
   - `%APPDATA%\rk_wcfr\config.json` 内容发生变化
   - 无需重启进程即可生效（若端口变更，需用新端口访问）

## 场景 4：预览与摄像头权限
1) 进入“预览”页
2) 预期：
   - 看到画面（若无画面：按“打开隐私设置”检查 Windows 摄像头权限）

## 场景 5：注册与清空库
1) 进入“预览”页
2) 在“注册 personId”输入框输入一个名字（例如 `alice`）
3) 点击“注册”
4) 点击“清空库”
5) 预期：
   - 注册与清空动作均返回成功提示（如失败，页面显示明确错误信息）

## 场景 6：IE11 降级（可选）
1) 使用 IE11 打开 `http://127.0.0.1:<port>/`
2) 预期：
   - 跳转到降级提示页（`ie11.html`）
   - 给出“请使用 Edge/Chrome/Firefox”的提示，不出现白屏或崩溃
