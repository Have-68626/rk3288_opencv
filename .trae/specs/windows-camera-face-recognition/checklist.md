# Checklist

- [x] Windows 摄像头枚举/连接可用（MF/DS 其一完整落地）
- [x] 支持 USB/内置摄像头识别与分辨率配置（640x480~1920x1080）
- [x] 预览帧率 ≥30FPS（在定义测试条件下可复现）
- [x] 支持相机切换、分辨率调整、画面翻转
- [x] 人脸检测 + 特征提取 + 比对识别流程完整（单人/多人）
- [x] 识别准确率 ≥95%（在定义数据集与口径下可复现）
- [x] UI 实时叠加框/姓名/ID/置信度
- [x] 识别日志落盘（时间戳/结果/置信度/错误原因），并可离线解析
- [x] CPU ≤30%、内存 ≤500MB（在定义测试条件下可复现），异常不崩溃且可恢复
- [x] 单元测试覆盖核心识别逻辑并可在 CI/本机运行
- [x] 性能测试报告包含不同光照条件下的识别效果与资源曲线
- [x] 文档齐全：编译文档、依赖清单、用户操作手册（ZH_CN），且无硬编码路径
