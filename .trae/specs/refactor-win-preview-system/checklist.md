# Checklist

- [x] UI 布局响应式：窗口缩放时控件不重叠、不溢出
- [x] 预览区保持宽高比且不拉伸变形（裁剪填充/letterbox 均可用）
- [x] 分辨率下拉包含 1080p/720p/480p，FPS 支持 15/30/60
- [x] 分辨率/FPS 组合验证与失败回滚可用，失败原因提示清晰
- [x] 切换重启耗时统计可复现，目标 <300ms（按口径）
- [x] 轻量检测模型可用（SSD MobileNet/DNN），每帧推理 + 坐标映射正确
- [x] 叠加绘制符合样式（半透明白描边 2px、圆角 5px、置信度百分比）
- [x] 性能监控可用：延迟>100ms 或丢帧率>5% 时自动降级并可恢复
- [x] 本地输出端口可用：GET /api/faces 返回最新 JSON；SSE（若启用）可订阅
- [x] 外部 POST 推送可用：可配置 URL、节流、退避重试，不阻塞预览
- [x] 测试 A/B/C 均有可复现步骤与报告产出
- [x] 文档更新完成（用户手册/DEVELOP/CREDITS），无硬编码路径
